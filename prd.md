## 🐾 页面结构与业务需求分析

### 1. 核心产品定位
**加密货币短期价格预测市场（Prediction Market）** — 用户可以对 BTC 在极短时间窗口（如5分钟、1小时）内的涨跌方向进行押注。类似于"合法期货/期权"的极简版。

### 2. 页面核心模块

| 模块 | 功能说明 | 技术要点 |
|------|---------|---------|
| **顶部导航栏** | Logo、市场分类（热门/政治/体育/加密/电竞等）、登录/注册 | 多语言(i18n)、用户认证、响应式 |
| **市场标题区** | 显示"比特币上涨或下跌5个月" + 时间窗口（如 6月5日 04:00-04:05 ET） | 动态时间倒计时、市场状态(open/closed) |
| **实时价格区** | 目标价格展示（$63,052.71）+ "前往实时盘口"按钮 | **Chainlink BTC/USD 实时价格流** |
| **涨跌图表** | 双栏Canvas图表（Up vs Down），类似K线/蜡烛图 | TradingView Lightweight Charts 或类似图表库 |
| **订单簿** | 买卖盘口深度展示 + 成交量（$36） | 订单匹配引擎、实时WebSocket推送 |
| **时间线** | 横向时间轴（04:00 → 04:05 → 04:10...），可翻到过去场次 | 时间轮转、历史市场切换 |
| **市场规则** | 结算规则说明 + Chainlink数据源声明 | 规则配置系统、可自定义结算源 |
| **评论区** | 发表评论、按时间排序、持仓者标签 | 评论系统、用户画像、社交互动 |
| **顶级持仓者/持仓/动态** | 三个Tab：大户持仓分布、个人持仓、市场动态 | 透明度系统、链上持仓验证 |

### 3. 关键业务需求

#### 🎯 A. 市场引擎（最核心）
- **自动创建市场**：按时间窗口自动生成涨跌二元期权市场
- **结算逻辑**：链上Chainlink预言机自动结算（时间窗口结束价 vs 开始价）
- **订单匹配**：买Up/Down token的CLOB（中央限价订单簿）
- **代币模型**：Up token = Yes（涨），Down token = No（跌），价格0-1美元

#### 💰 B. 交易系统
- **下注UI**：输入金额 → 购买Up/Down token
- **钱包集成**：USDC入金/出金，链上结算（Polygon链）
- **持仓管理**：当前持仓、历史盈亏、PnL追踪

#### 📊 C. 实时数据
- **价格流**：Chainlink BTC/USD 实时数据（关键！这是结算源）
- **图表**：实时K线/蜡烛图 + Up/Down token价格走势
- **WebSocket推送**：订单簿、成交、价格变动全部实时

#### 👥 D. 用户系统
- **钱包登录**：MetaMask/OKX Wallet 等Web3钱包
- **用户画像**：链上地址匿名 + 自定义用户名
- **持仓者排行榜**：透明展示大户持仓方向

#### ⏰ E. 时间轮转系统
- **5分钟市场**：每5分钟自动开一个新市场，24/7不间断
- **倒计时**：显示当前市场剩余时间
- **历史归档**：过去市场可回看结果

#### 🏗️ F. 后台/运营
- **市场创建规则引擎**：可配置资产（BTC/ETH等）、时间窗口（1m/5m/1h等）、结算源
- **风控系统**：异常交易检测、最小/最大下注额
- **收入模型**：交易手续费（通常0% maker / 0% taker，靠市场做市盈利）

### 4. 技术栈建议

```
前端：Next.js + TailwindCSS + TradingView Lightweight Charts
后端：Node.js/Python + PostgreSQL + Redis
实时：WebSocket (Socket.io) + Chainlink 喂价
链上：Polygon (EVM) + CTF/CLOB合约
图表：TradingView Charting Library 或 lightweight-charts
部署：Vercel + Railway/云服务器
```

### 5. 开发优先级

1. **P0 - MVP核心**：市场创建引擎 + 结算逻辑 + 基本下单UI + Chainlink价格接入
2. **P1 - 交易体验**：订单簿 + 实时图表 + 钱包连接 + 持仓管理
3. **P2 - 社交/增长**：评论区 + 持仓者排行 + 分享功能
4. **P3 - 扩展**：更多资产(ETH/SOL) + 更多时间窗口 + 移动端App

### 6. 💡 差异化机会

Polymarket是通用预测市场，这个BTC涨跌页只是其中一个品类。如果你想做一个**专注加密短期预测**的产品，可以考虑：
- 支持更多时间窗口（1分钟到24小时）
- 支持更多资产和价差
- 加入社交跟单功能（跟大户持仓）
- 简化UX（不需要Web3钱包，法币直接入金）
- 加入AI趋势分析提示
